Notable Projects

Project titles and descriptions

[P07] A Mars Exploration Rover Prototype 

Project venue:  Indian Rover Challenge 2018(IRC), Vellore, India.

Description:  Its an off-road vehicle with a robotic manipulator. This rover can perform different manipulation task with its robotic arm. 

[P06] Updated Mars Exploration Rover Prototype

Project venue:  European Rover Challenge 2018 (ERC), Starachowice, Poland 

Description:  This rover prototype is an off-road remote operated vehicle and capable of performing science, maintenance, navigation, and probing task. This robot can analyze a mars like environment by testing the humidity, PH and type of gas present in the environment. An on board robotic manipulator can pick and place different objects into robot body for further analysis.

[P05] Fighting robot

Project venue:  Techfest-2017, Mumbai, India 

Description:  This robot is made for robofight competition. Here, this robot attack its opponet robot with its mechanical spinner and lifter. Its a four wheeler drive RC robot.  

[P04] Writing robotic arm

Project venue:   "Esho Robot Banai (lets make robot)" Reality Show 2019. Streaming on: Channel I. 

Description:  This robot can draw different shapes on a plane surface. Also, an android application is developed with which the robot can mimic the drawing on an android screen.  

[P03] Smart Driving Assistant System 

Project venue:  "Esho Robot Banai (lets make robot)" Reality Show 2019. Streaming on: Channel I. 

Description:  This system detect the drowsy face of a driver and trigger an alarm. Besides, the position of a vehicle can easily be determined from the operating station. Also, it takes care of overloading condition. If a truck is overloaded, the engine will not start. 

[P02] Power factor corrector

Project venue:  Undergraduate project, 2017.

Description:  It can measure the power factor and correct it with different settings of a capacitor bank.
